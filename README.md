# AeroMorph 3D

AeroMorph is a CAD-focused geometry morphing project exploring how engineering models can be programmatically modified while preserving CAD usability.

## Goal

Create a system that can:
- Import engineering CAD models
- Display and inspect geometry
- Select regions/features
- Apply controlled deformations
- Export modified CAD files usable in engineering workflows

## Current Focus

Learning and experimenting with:
- CAD geometry representations
- OpenCASCADE through build123d
- Parametric modeling
- Geometric transformations

## Tech Stack

- Python
- build123d / OpenCASCADE
- PyVista
- NumPy

## Status

Early development.
